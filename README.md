# Budget & Ledger  

This repository contains a structured budgeting and ledger system. Version 13 is the current stable architecture lock.  

## Overview  
- **Architecture:** See [architecture/v13.md](./architecture/v13.md) for the complete design, money flow model, governance rules, and system constraints.  
- **Operating Guide:** See [operating/v13.md](./operating/v13.md) for instructions on using the workbook, recording income, expenses, transfers, refunds, and closing each month.  
- **Change Log:** See [CHANGELOG.md](./CHANGELOG.md) for a record of all notable changes across versions.  
- **Release Notes:** See [releases/v13/RELEASE_NOTES.md](./releases/v13/RELEASE_NOTES.md) for details of the v13 release.  

Version 13 is architecture-locked; any structural changes will be introduced in v14. The roadmap for v14 is available under [roadmap/v14.md](./roadmap/v14.md).
